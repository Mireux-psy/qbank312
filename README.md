# 312心理学考研题库 - GitHub Pages 部署指南

> 本文档手把手教你把题库网页部署到 GitHub Pages，让手机可以像 App 一样使用。

---

## 第一步：新建 GitHub 仓库

1. 打开 [github.com](https://github.com) 并登录你的账号
2. 点击页面右上角 **+** 号，选择 **New repository**（新建仓库）
3. 填写信息：
   - **Repository name**（仓库名）：填写 `qbank312`（或你喜欢的名字，只用英文和数字）
   - **Description**（描述）：可填 `312心理学考研题库`
   - 选择 **Public**（公开）
   - 勾选 **Add a README file**（添加 README）
   - 点击最下方的绿色按钮 **Create repository**

---

## 第二步：上传文件

### 方法一：网页直接上传（推荐新手）

1. 进入你刚创建的仓库页面
2. 点击页面上方的 **Add file** 按钮，选择 **Upload files**
3. 将 `index.html` 文件拖拽到页面中间的虚线框里，或点击 **choose your files** 选择文件
4. 页面下方会出现一个 **Commit changes** 区域：
   - 在第一个输入框里填写：`添加题库网页`
   - 点击绿色按钮 **Commit changes**
5. 等待上传完成，页面上会出现 `index.html` 文件

---

## 第三步：开启 GitHub Pages

1. 在你的仓库页面，点击顶部的 **Settings**（设置）标签
2. 左侧菜单往下滑，找到并点击 **Pages**
3. 在 **Source** 部分：
   - Branch 下拉菜单选择 **main**
   - 点击 **Save**
4. 页面会出现一行绿色提示：`Your site is ready to be published at https://你的用户名.github.io/qbank312/`
5. **等待 1~3 分钟**，然后刷新这个 Pages 页面
6. 当提示变成 `Your site is live at https://你的用户名.github.io/qbank312/` 时，点击链接即可访问

---

## 第四步：手机添加到主屏幕

### iPhone (Safari)

1. 用 Safari 打开你的网址（如 `https://你的用户名.github.io/qbank312/`）
2. 点击底部中间的 **分享** 按钮（方框带箭头的图标）
3. 往下滑，找到 **添加到主屏幕**，点击
4. 点击右上角 **添加**
5. 桌面上会出现 "312题库" 图标，点击即可全屏打开

### Android (Chrome)

1. 用 Chrome 打开你的网址
2. 页面底部可能会自动弹出"安装到桌面"提示，点击 **安装**
3. 如果没有弹出，点击右上角 **三个点菜单** → **添加到主屏幕**
4. 桌面上会出现图标

---

## 第五步：备份/迁移数据

1. 在电脑浏览器中打开题库网页
2. 点击工具栏上的 **导出备份**，下载 `.json` 文件
3. 在手机浏览器中打开同一个网址
4. 点击工具栏上的 **导入备份**，选择刚才下载的 `.json` 文件
5. 数据就会同步到手机上

> 提示：以后每次添加新题目后，记得在电脑上导出备份，在手机上导入一次即可同步。

---

## 常见问题

**Q：页面打开后一片空白？**
A：GitHub Pages 部署需要等待 1~3 分钟，请刷新重试。

**Q：网址太长记不住？**
A：部署完成后，你可以把网址收藏到浏览器书签，或者直接添加到手机主屏幕，以后点图标就能打开。

**Q：数据存在哪里？**
A：数据存在浏览器的本地数据库中（IndexedDB），不会上传到 GitHub，完全私密。换设备时需要手动导出/导入备份。
