# 312心理学考研题库 & 错题本

一款纯前端心理学考研题库工具，支持题目录入、错题管理、好题收藏、刷题练习和统计分析。通过 GitHub Pages 部署，手机添加到主屏幕后可像 App 一样使用。

在线访问：**https://mireux-psy.github.io/qbank312/**

---

## 功能一览

### 题目管理
- 支持 **312统考四类题型**：单选题、多选题、简答题、论述题（综合）
- 题目自动拆分：粘贴"题目+选项"文本时，系统自动识别 A/B/C/D 选项（大小写均可，支持多行和单行格式）
- 支持图片作为题目（直接拍照/上传图片，无需手动打字）
- 选项可手动输入，支持 `A.` / `A、` / `A)` / `A-` / `|` 分隔等多种格式
- 每题可标记：科目、来源、难度、考察知识点、标签
- 支持批量导入（文本、CSV、图片）和导出备份（JSON）

### 错题本 & 好题集
- 标记错题（红色标签）和好题（星标收藏）
- 独立的"好题集"页面，方便集中复习收藏的题目
- 筛选和搜索：按科目、题型、难度、关键词快速定位

### 刷题模式
- 翻卡片式刷题，点击翻转查看答案
- 按科目/题型/难度筛选题池
- 标记"已掌握"或"没掌握"，实时追踪复习进度
- 题池清空时提示完成一轮复习

### 统计分析
- 总题数、错题数、好题数、待复习数、已掌握数概览
- 各科目题目分布柱状图
- 错误率排行（薄弱知识点）
- 高频知识点统计

### 数据安全
- 所有数据存储在浏览器本地（IndexedDB），不会上传到服务器
- 支持导出 JSON 备份文件，换设备时导入即可同步
- 离线可用，无需联网

## 手机端使用

### 添加到主屏幕
- **iPhone**：Safari 打开网址 → 分享按钮 → 添加到主屏幕
- **Android**：Chrome 打开网址 → 三个点菜单 → 添加到主屏幕

### 数据同步
1. 在电脑浏览器中导出备份（工具栏 → 导出备份）
2. 在手机浏览器中导入备份（工具栏 → 导入备份）

## 选项自动识别格式

粘贴题目时，系统会自动拆分选项到选项区。支持的格式：

| 格式 | 示例 |
|------|------|
| 多行（推荐） | 每行一个选项，以 `A.` / `A、` / `A)` 等开头 |
| 单行 | `题目 A. xxx B. xxx C. xxx D. xxx` |
| 管道符 | `选择性\|整体性\|理解性\|恒常性` |

- 大小写均可（a./A. 都支持）
- 至少出现 2 个选项才自动识别，避免题目中普通字母被误判
